# angular-helloworld

1. 環境建置
- 安裝Node.js
- 安裝Visual Studio Code
- 安裝Angular CLI

    Power Shell執行: npm install -g @angular/cli

2. 創建HelloWorld

- 專案目錄下執行指令: ng new HelloWorld

- 啟動專案: npm start

    瀏覽器輸入http://localhost:4200/
